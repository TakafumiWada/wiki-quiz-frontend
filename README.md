# wiki-quiz-frontend

## 概要

wiki-quiz のフロントエンドです。バックエンドで取得したクイズデータを表示し、実際に操作できます。

## 環境

```txt
vue : ^2.6.11
vue-router : ^3.2.0
vuex : ^3.4.0
```
